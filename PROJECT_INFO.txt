ПРОЕКТ: Поиск шаблонов форм
=====================================

ОПИСАНИЕ:
CLI приложение для поиска шаблонов форм в базе данных на основе переданных полей.

СТРУКТУРА ПРОЕКТА:
- app.py              - Основной файл CLI приложения
- database.py         - Модуль для работы с базой данных (TinyDB)
- validators.py      - Модуль для валидации типов полей
- init_db.py         - Скрипт для инициализации тестовой БД
- test_db.json       - Файл тестовой базы данных
- test_app_examples.py - Скрипт для проверки работы приложения (примеры использования)
- requirements.txt   - Зависимости проекта
- README.md          - Подробная документация

ТРЕБОВАНИЯ ТЗ - ВЫПОЛНЕНО:
✓ 1. Использована база данных TinyDB
✓ 2. Версия Python: 3.6+ (протестировано на Python 3.11)
✓ 3. Файл с тестовой БД (test_db.json) включен
✓ 4. README.md с инструкциями по установке и запуску
✓ 5. Готово к выгрузке на GitHub

УСТАНОВКА И ЗАПУСК:
1. pip install -r requirements.txt
2. python init_db.py (если БД еще не создана)
3. python app.py get_tpl --имя_поля=значение

ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ:
python app.py get_tpl --f_name1=vasya@pukin.ru --f_name2=27.05.2025
python app.py get_tpl --phone="+7 909 996 26 79" --email=test@example.com --message=123

ПРОВЕРКА РАБОТЫ ПРИЛОЖЕНИЯ:
python test_app_examples.py
